Complex vector spaces#

Eigenvalues are often complex, making it essential to consider eigenvalue problems within the setting of complex vector spaces. This section briefly outlines key distinctions in handling complex spaces.

Inner products and norms#

For complex spaces, the Euclidean inner product is defined as:

\[ \langle x, y \rangle = \overline{y}^T x = \sum_j x_j \overline{y_j}. \]

This requires taking the complex conjugate of \(y\) in the inner product calculation.

The 2-norm of a vector is hence:

\[ \|x\|_2 = \left(\sum_{j} x_j \overline{x_j}\right)^{1/2} = \langle x, x \rangle^{1/2}. \]

Hermitian matrices#

In complex spaces, the counterpart to transposed matrices is the conjugate transpose:

\[ A^H := \overline{A}^T. \]

With this, the following relation holds:

\[ \langle Ax, y \rangle = \langle x, A^H y \rangle. \]

A matrix is Hermitian (or self-adjoint) if:

\[ A^H = A. \]

Consequently, the diagonal elements of a Hermitian matrix are real-valued.

Unitary matrices#

Reflecting the altered inner product definition, the concept of orthogonal matrices in complex spaces changes to unitary matrices. A matrix \(Q \in \mathbb{C}^{n \times n}\) is unitary if:

\[ Q^H Q = I. \]

Thus, a unitary matrix in complex spaces is analogous to an orthogonal matrix in real spaces. All unitary matrices satisfy

\[ |\det(Q)| = 1. \]

Python skills#

Working with complex matrices is well supported in Python.

import numpy as np

# Step 1: Create complex matrices A and B
A = np.array([[1 + 2j, 3 + 4j], [5 + 6j, 7 + 8j]])
B = np.array([[1, 1j], [1j, 1]] / np.sqrt(2))

# Step 2: Matrix multiplication
product = np.dot(A, B)

# Step 3: Conjugate transpose (Hermitian) of A and B
A_H = np.conjugate(A.T)
B_H = np.conjugate(B.T)

# Step 4: Check if A or B are unitary
is_unitary_A = np.allclose(np.dot(A_H, A), np.eye(A.shape[0]))
is_unitary_B = np.allclose(np.dot(B_H, B), np.eye(B.shape[0]))

# Displaying the results
print("Matrix A:\n", A)
print("Matrix B:\n", B)
print("Product of A and B:\n", product)
print("Conjugate transpose (Hermitian) of A:\n", A_H)
print("Conjugate transpose (Hermitian) of B:\n", B_H)
print("Is Matrix A unitary?:", is_unitary_A)
print("Is Matrix B unitary?:", is_unitary_B)

Self-check questions#

Question

Show that for a Hermitian matrix \(A \in \mathbb{C}^{n\times n}\) we have

\[ x^H A x \in\mathbb{R} \qquad \forall \, x\in\mathbb{C}^{n}. \]

A Hermitian matrix is called positive definite if

\[ x^H A x > 0 \qquad \forall \, x\in\mathbb{C}^{n} \setminus \{0\}. \]

Show that if a Hermitian matrix \(A\) is positive definite, then all eigenvalues of \(A\) are real and positive

Answer

Using that \(A\) is Hermitian (\(A=A^H\)) and that \(x^HAx = x^HAx^T \in \mathbb{C}\), we find

\[ \overline{x^HAx} = \left(x^HAx\right)^T = x^HA^Hx = x^HAx. \]

Therefore \(x^H A x \in \mathbb{R}\) for all \(x \in \mathbb{C}^n\).

Let \(\lambda_j\) be an eigenvalue with associated eigenvector \(x_j\) and \(\|x_j\|_2 = 1\). For \(A\) positive definite, we have that \(\lambda_j = x_j^H A x_j > 0\).

Question

Show that all unitary matrices \(Q\) satisfy \(|\det(Q)| = 1\).

Answer

From \(Q^H Q = I\), we conclude

\[ 1 = \det(I) = \det(Q^H Q) = \det(Q^H) \det(Q) = \overline{\det(Q)} \det(Q) = |\det(Q)|^2. \]

It follows that \(|\det(Q)| = 1\).
