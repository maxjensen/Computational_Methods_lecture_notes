# Stable Quadrature Schemes

## Composite Quadratures

When dealing with functions $f$ that may lack smoothness, it can be advantageous to divide the integration interval $[a,b]$ into $m$ smaller, equally sized subintervals $\{[x_{i-1},x_i]\}_{i=1}^m$. Each subinterval has a length of $h:=(b-a)/m$. Within each subinterval, we apply a low-degree interpolatory quadrature formula.

For example, using the trapezium rule on $[x_{i-1},x_i]$, we find

$$
\begin{align}
   \mathcal{C}_{1,m}(f) &= \frac{b-a}m\, \Bigl[ \Bigl( \frac{1}{2} f(x_0) + \frac{1}{2} f(x_1) \Bigr)
	+ \Bigl(\frac{1}{2} f(x_1) + \frac{1}{2} f(x_2) \Bigr) + \cdots
	+ \Bigl( \frac{1}{2} f(x_{m-1}) + \frac{1}{2} f(x_m) \Bigr) \Bigr] \notag\\
   &= h \Bigl[ \frac{1}{2} f(x_0) + \sum_{i=1}^{m-1} f(x_i) + \frac{1}{2} f(x_m) \Bigr].
\end{align}
$$

This scheme involves $m+1$ evaluations of $f$. To use Simpson's rule, we introduce an additional midpoint node $x_{i-1/2}:=(x_{i-1}+x_i)/2$ in each subinterval. The composite Simpson's rule is then given by:

$$
\begin{align}
   \mathcal{C}_{2,m}(f) &= \frac{b-a}{m} \, \Bigl[ \frac{1}{6} f(x_0) + \frac{2}{3} f(x_{1/2}) + \frac{1}{6} f(x_1)
	+ \frac{1}{6} f(x_1) + \frac{2}{3} f(x_{3/2}) + \frac{1}{6} f(x_2)\\
	& \phantom{= \frac{b-a}{m} \, \Bigl[} + \cdots + \frac{1}{6} f(x_{m-1}) + \frac{2}{3} f(x_{m-1/2}) + \frac{1}{6} f(x_m)\Bigr]\\
	&= \frac{h}{6} \Bigl[ f(x_0) + 4 \sum_{i=1}^{m} f(x_{i-1/2}) + 2 \sum_{i=1}^{m-1} f(x_i) + f(x_m) \Bigr].
\end{align}
$$

It involves $2m+1$ evaluations of $f$.

Error bounds for these methods can be obtained directly from those of the underlying methods. For $\mathcal{C}_{1,m}$ we have, in the subinterval $[x_{i-1},x_i]$,

$$
\bigl|\mathcal{I}(f)-\mathcal{I}_1(f)\bigr| \le \frac{1}{2!}\,\max_{\xi\in[x_{i-1},x_i]}\,|f''(\xi)| \,\int_{x_{i-1}}^{x_i} |\omega_2(x)| dx = \frac{h^3}{12}\,\max_{\xi\in[x_{i-1},x_i]}\,|f''(\xi)|,
$$

where $\omega_2(x)=(x-x_{i-1})(x-x_i)$. Adding these up, we have in $[a,b]=[x_0,x_m]$,

$$
\bigl|\mathcal{I}(f)-\mathcal{C}_{1,m}(f)\bigr|	\le \frac{mh^3}{12}\,\max_i\Bigl\{\max_{\xi\in[x_{i-1},x_i]}\,|f''(\xi)|\Bigr\} = (b-a)\,\frac{h^2}{12}\,\max_{\xi\in[a,b]}\,|f''(\xi)|.
$$

## Gaussian Quadratures

The degree of exactness of an interpolatory quadrature $\mathcal{Q}_n(f)$ is at least $n$. By considering $f(x)=(x-x_0)^2\cdots(x-x_n)^2$, we find that the degree of exactness of any interpolatory quadrature is no more than $2n+1$. Gauss showed that by judicious choice of the nodes $\{x_k\}$, one can indeed obtain a quadrature exact for all $f \in \mathcal{P}_{2n+1}$. The resulting Gaussian quadrature rests on the following:

```{prf:theorem} Gaussian quadrature
The interpolatory quadrature formula $\mathcal{G}_n$ achieves a degree of exactness $2n+1$ if and only if its nodes $x_0, \ldots, x_n$ are such that the polynomial

$$
\tilde{p}(x) = \prod_{i=0}^n (x - x_i),
$$

meets the orthogonality condition $\langle \tilde{p}, p \rangle = 0 \quad \forall \, p \in \mathcal{P}_k$.
```

Thus, the nodes $\{x_k\}_{k=0}^n$ are the roots of a polynomial $\phi_{n+1} \in \mathcal{P}_{n+1}$ which is in the orthogonal complement relative to the $(n+1)$-dimensional space $\mathcal{P}_n$ within the $(n+2)$-dimensional space $\mathcal{P}_{n+1}$ with respect to the inner product $\langle p, q \rangle = \int_a^b p(x) \, q(x) \, dx$.  A proof of the theorem is given in the optional materials section.

```{prf:definition}
An interpolatory quadrature method that achieves a degree of exactness $2n+1$ is termed a *Gaussian quadrature method*. 
```

Once the nodes are determined, the coefficients can be computed as usual. Unlike the coefficients $\sigma_k$ of a general quadrature formula, Gaussian quadrature weights are always positive. This can be illustrated by considering $\bar p_k(x)=\hat p(x)^2/(x-x_k)^2 \in \mathcal{P}_{2n}$. With the degree of exactness of  $2n+1$ and $\bar p_k(x_j) = \delta_{jk}$, we find that:

$$
0 < \mathcal{I}(\bar p_k) = \mathcal{Q}(\bar p_k) = \sigma_k.
$$

## Python skills

### Composite Newton-Cotes quadrature rules

Scipy provides an implementation of the composite trapezoidal and Simpson's rules, Numpy only has the former.

```python
import numpy as np
from scipy import integrate

# Define a function to integrate
def my_function(x):
    return x ** 2

# Define the interval
a = 0  # Lower limit
b = 1  # Upper limit
n = 100  # Number of sample points

# Generate sample points
x = np.linspace(a, b, n)
y = my_function(x)

# Trapezoidal Rule
trapz_result_np = np.trapz(y, x)
trapz_result_scipy = integrate.trapz(y, x)

# Simpson's Rule
simps_result = integrate.simps(y, x)

# Display the results
print("Trapezoidal Rule (numpy):", trapz_result_np)
print("Trapezoidal Rule (scipy):", trapz_result_scipy)
print("Simpson's Rule:", simps_result)
```

### Gaussian quadrature

The code below compares exact integration with SciPy's Gaussian quadrature implementation `integrate.quad`, which uses an adaptive composite formula. In addition, the Gaussian quadrature function `integrate.fixed_quad` with a specified number of quadrature points is evaluated.

```python
import numpy as np
import scipy.integrate as integrate

# Define a function to integrate and its exact integral
def my_function(x):
    p = 9
    return x**p

def integral_of_my_function(a,b):
    p = 9
    return (b**(p+1) - a**(p+1)) / (p+1)

# Define the interval
a = 1  # Lower limit
b = 5  # Upper limit

# Using exact integration formula
result_exact = integral_of_my_function(a,b)
print("Exact integration:", result_exact)

# Using scipy.integrate.quad, which uses an (adaptive) composite Gaussian quadrature
result_quad, _ = integrate.quad(my_function, a, b)
print("Gaussian Quadrature (scipy.integrate.quad):", result_quad)

# Using scipy.integrate.fixed_quad with different orders
for n in [2, 3, 4, 5]:  # Different number of quadrature points
    result_fixed_quad, _ = integrate.fixed_quad(my_function, a, b, n=n)
    print(f"Gaussian Quadrature of order {n} (fixed_quad):", result_fixed_quad)
```

## Optional material

`````{dropdown} Theory of Gaussian quadrature

We consider the more general integration problem

$$
\mathcal{I}(f) := \int_a^b f(x) \, \gamma(x) \, dx, \qquad f \in C[a,b]
$$

What is the relevance of $\gamma$? 

1. In applications, one often has to integrate functions $g(x)$ which can be described as the product of a function $\gamma$ which one can easily integrate exactly and a function $f$ which is hard to integrate but smooth: $g = f \cdot \gamma$. Then a widely used approach is to approximate $f$ by a polynomial $p$ and to integrate $p\cdot \gamma$ instead.

2. We shall now allow unbounded integration domains, i.e. $a \in [\infty, \infty)$ and $b \in (-\infty, \infty]$. Then $\gamma$ is chosen such that $p \cdot \gamma$ is integrable for $p \in \mathcal{P}_n$.

At first reading, it is recommended to think of the special case $\gamma \equiv 1$ and $a, b \in \mathbb{R}$.

### Preparation for Gaussian Quadrature

It is convenient to expose the dependence on the integration interval directly in the quadrature formula:

$$
\mathcal{G}_n(f) = (b-a) \sum_{k=0}^n \sigma_k f(x_k).
$$

Then the quadrature weights and nodes that are determined for a reference interval can be easily transferred to general intervals $[a,b]$.

When examining Newton-Cotes formulas we did not use that we can freely choose the nodes $x_k$.

We shall now see that one can construct much better quadrature formulas by systematically searching for the best locations for the nodes. Recall that we already showed that the degree of a quadrature formula $\mathcal{G}_n$ with $n+1$ nodes is at most $2n + 1$.

We will find out that one can construct quadrature methods with a degree of exactness $2 n + 1$. The mathematical tool which allows us to construct such methods is the scalar product

$$
\langle f,g \rangle_{a,b,\gamma} = \int_a^b f(x) g(x) \, \gamma(x) \, dx.
$$

The definition of a scalar product was given in the optional materials section of the least squares approximation chapter. Convince yourself that $\langle \cdot, \cdot \rangle_{a,b,\gamma}$ is indeed a scalar product. Here $\gamma$ an integration weight, meaning that for our purposes that $\gamma$ is piecewise continuous, integrable and positive. Typical examples are $\gamma = 1$, $a,b \in \mathbb{R}$ and $\gamma = 1/t$, $a=0$, $b=\infty$ and $\gamma = 1/\sqrt{1-t^2}$, $a=-1$, $b=1$.

Our analysis aims to show that the best quadrature nodes coincide with the roots of orthogonal polynomials. First we need to establish two results about orthogonal polynomials.

````{prf:theorem}
Let $\tilde{p} \in \mathcal{P}_{k+1}$, $k \ge 0$, and

```{math}
:label: orthcond
\langle \tilde{p}, p \rangle = 0 \qquad \forall \, p \in \mathcal{P}_k.
```

Then either $\tilde{p} = 0$ or $\tilde{p}$ has exactly $k+1$ distinct roots in the open interval $(a,b)$.
````

```{prf:proof}
Suppose that $\tilde{p} \neq 0$. Let $\xi_0, \xi_1, \ldots, \xi_\ell$ be the roots of $\tilde{p}$ at which $\tilde{p}$ changes its sign. Let

$$
s(x) := \prod_{j=0}^\ell (x - \xi_j)
$$

if such roots exist. Otherwise we set $s(x) = 1$. The polynomial $s$ changes its sign exactly at the $\xi_0, \xi_1, \ldots, \xi_\ell$ as well. The reason is that at each $\xi_i$, $0 \le i \le \ell$, exactly one factor in the product form of $s(x)$ changes its sign. Therefore, the product $\tilde{p} \cdot s$ has no sign changes at all. Thus

$$
\langle \tilde{p}, s \rangle_{a,b,\gamma} \neq 0.
$$

By {eq}`orthcond`, the degree of $s$ is larger than $k$. In other words $\ell > k$, which means that $\tilde{p}$ has more than $k$ sign changes in $(a,b)$. However, because $\tilde{p} \in \mathcal{P}_{k+1}$ has at most $k+1$ sign changes.
```

It will become apparent that the roots of $\tilde{p}$ are the best possible quadrature nodes. However, wait. Do we know if there is only one polynomial which satisfies {eq}`orthcond`. Different polynomials satisfying {eq}`orthcond` could have different roots. The next theorem ensures that this is not a problem. It says that condition {eq}`orthcond` determines the polynomial $\tilde{p}$ up to a multiplicative factor. And polynomials which only differ by a multiplicative factor have the same roots (unless one of them is $0$). (We could also argue as in the main text that $\tilde{p}$ is in the $1$-dimensional orthogonal complement of $\mathcal{P}_n$ within $\mathcal{P}_{n+1}$; however, the less abstract proof below may be illuminating.)

```{prf:theorem}
Suppose $\tilde{p}, \bar{p} \in \mathcal{P}_{k+1} \setminus \{ 0 \}$ satisfy {eq}`orthcond`. Then there is an $\alpha \in \mathbb{R}$ such that $\bar{p} = \alpha \, \tilde{p}$ and $\alpha \neq 0$.
```

```{prf:proof}
Let

$$
\tilde{p}(x) &=& \tilde{a}_0 + \tilde{a}_1 x + \ldots \tilde{a}_{k+1} x^{k+1},\\
\bar{p}(x) &=& \bar{a}_0 + \bar{a}_1 x + \ldots \bar{a}_{k+1} x^{k+1}.
$$

The coefficient $\tilde{a}_{k+1} \neq 0$, because otherwise $\tilde{p} \in \mathcal{P}_k$. However then from {eq}`orthcond` it follows $\langle \tilde{p}, \tilde{p} \rangle = 0$ implying that $\| \tilde{p} \| = 0$ which is not possible because it was assumed that $ \tilde{p} \neq 0$. Therefore we may set $\alpha = \bar{a}_{k+1} / \tilde{a}_{k+1}$.

Let $\Delta p = \alpha \tilde{p} - \bar{p}$. Because of the choice of $\alpha$ the difference $\Delta p$ belongs to $\mathcal{P}_k$, that is the highest-order coefficient cancels out. With {eq}`orthcond` we conclude

$$
\| \Delta p \|^2 = \langle \Delta p, \Delta p \rangle = \langle \alpha \tilde{p} - \bar{p}, \Delta
p \rangle = \alpha \langle \tilde{p}, \Delta p \rangle - \langle \bar{p}, \Delta p \rangle = 0.
$$

Therefore $\| \Delta p \| = 0$. This implies that $\bar{p} = \alpha \, \tilde{p}$.
```

```{prf:example}
With the Gram-Schmidt method we compute an orthonormal basis of $\mathcal{P}_2$ with respect to the scalar product:

$$
\langle f,g \rangle_{-1,1,\gamma} = \int_{-1}^1 f(x) g(x) \, dx, \qquad \gamma = 1.
$$

The basis functions are:

$$
q_0 = \textstyle \frac{1}{\sqrt{2}},\qquad q_1 = \sqrt{\textstyle\frac{3}{2}} t,\qquad q_2 =
\sqrt{\textstyle\frac{45}{8}} (t^2 - \textstyle\frac{1}{3}).
$$

The polynomial $q_2$ satisfies {eq}`orthcond` for $k=1$: Every polynomial $p$ in $\mathcal{P}_1$ can be written as a linear combination of $q_0$ and $q_1$:

$$
p = \alpha q_0 + \beta q_1, \qquad \alpha, \beta \in \mathbb{R}.
$$

Noting that $q_0, q_1$ and $q_2$ are orthogonal with respect to each other we find

$$
\langle q_2, p \rangle = \langle q_2, \alpha q_0 + \beta q_1 \rangle = \alpha \langle q_2,  q_0
\rangle + \beta \langle q_2, q_1 \rangle = 0.
$$

(In the same spirit one can examine the case $k=0$: One finds that $q_1$ satisfies {eq}`orthcond` for all $p \in \mathcal{P}_0$.)

As predicted by the theorem above, $q_2$ has two distinct roots in the interval $(-1,1)$. They are

$$
\pm \frac{1}{\sqrt{3}} \approx \pm 0.5774.
$$

All other quadratic polynomials which satisfy {eq}`orthcond` with $k = 1$ have, according to the last theorem, the form

$$
\delta (t^2 - 1/3)
$$

for some $\delta \in \mathbb{R}$.
```

We now come to the main theorem on Gauss quadrature. We may decide that the nodes of our favourite quadrature method are the roots of $\tilde{p}$.

```{prf:theorem}
Let $\gamma$ be an integration weight. The interpolatory quadrature formula $\mathcal{G}_n$ has the degree of exactness $2n+1$ with respect to $\gamma$ if and only if the nodes $x_0, \ldots, x_n$ of $\mathcal{G}_n$ are chosen such that

$$
\tilde{p}(x) = \prod_{i=0}^n (x - x_i)
$$

satisfies {eq}`orthcond`.
```

````{prf:proof}
a) Let $\mathcal{G}_n$ have the degree of exactness $2n+1$ with respect to $\gamma$. We need to show that then {eq}`orthcond` is satisfied. One observes

$$
\mathcal{G}_n (f) = \int_a^b f(x) \cdot \gamma(x) \, dx
$$

for all $f \in \mathcal{P}_{2n+1}$. This is in particular true for all $f := \tilde{p} \cdot p \in \mathcal{P}_{2n+1}$ with $p \in \mathcal{P}_{n}$. This means

$$
\int_a^b \tilde{p}(x) \, p(x) \, \gamma(x) \, dx = \mathcal{G}_n (\tilde{p} \cdot p)=
\sum_{i=0}^n \sigma_k (\underbrace{\tilde{p}(x_k)}_{=0} \cdot p(x_k)) = 0
$$

for all $p \in \mathcal{P}_{n}$. But this condition is precisely {eq}`orthcond`.

b) Now suppose {eq}`orthcond` holds. We have to show that $\mathcal{G}_n$ has the degree of exactness $2n +1$. Let $f \in \mathcal{P}_{2n+1}$. Then $f$ can be divided by $\tilde{p}$, giving

$$
f = \tilde{p} \cdot p + r
$$

where $p, r \in \mathcal{P}_k$. 

```{note}
We used here polynomial division: Given general $p_1 \in \mathcal{P}_\ell$ and $p_2 \in \mathcal{P}_j \setminus \{ 0 \}$ with $j \le \ell$, there are uniquely determined polynomials $q$ and $r$ such that $p_1 = q p_2 + r$ and such that the degree of $r$ is less then $j$. Moreover, the degree of $q$ plus $j$ must be equal to $\ell$.
```

Because {eq}`orthcond` holds, one observes

$$
\begin{eqnarray*}
\int_a^b f(x) \, \gamma(x) \, dx &=& \int_a^b (\tilde{p}(x) \cdot p(x) + r(x)) \, \gamma(x) \, dx\\
&=& \underbrace{\int_a^b \tilde{p}(x) \cdot p(x) \, \gamma(x) \, dx}_{=0} + \int_a^b r(x) \, \gamma(x) \, dx\\
&=& \int_a^b r(x) \, \gamma(x) \, dx.
\end{eqnarray*}
$$

On the other hand

$$
\begin{eqnarray*}
\mathcal{G}_n(f) &=& (b-a) \sum_{i=0}^n \sigma_k f(x_k) = (b-a) \sum_{i=0}^n \sigma_k
(\underbrace{\tilde{p}(x_k)}_{=0} \cdot p(x_k) + r(x_k))\\
& = & (b-a) \sum_{i=0}^n \sigma_k r(x_k) = \mathcal{G}_n(r).
\end{eqnarray*}
$$

Because $\mathcal{G}_n$ is an interpolatory quadrature formula it integrates $r$ exactly, i.e.

$$
\mathcal{G}_n(r) = \int_a^b r(x) \cdot \gamma(x) \, dx.
$$

In conclusion

$$
\int_a^b f(x) \, \gamma(x) \, dx = \int_a^b r(x) \, \gamma(x) \, dx = \mathcal{G}_n(r) =
\mathcal{G}_n(f).
$$

and $\mathcal{G}_n$ has the degree of exactness $2n+1$.
````

A method with degree of exactness $2n+1$ is called Gaussian quadrature method.

We finished the theory of Gauss quadrature. Let us now see what this means in practice.

```{admonition} Task
:class: tip
Given a fixed number of nodes $n+1$ and a positive, piecewise continuous,
integration weight $\gamma$, find the quadrature formula $\mathcal{G}_n$ with the degree of exactness $2n +
1$ with respect to $\gamma$.
```

**Motivation for this Task** Due to the high degree of exactness $2n + 1$ with respect to $\gamma$ this quadrature formula tends to give the best approximation to integrals of the type

$$
\int_a^b f(x) \, \gamma(x) \, dx
$$

where $f$ is a smooth function. (Ideally $f$ has infinitely many bounded derivatives. Generally, the less smooth $f$ is the larger the quadrature error becomes).

**Solution** The following steps have to carried out to construct $\mathcal{G}_n$:

1. In order to find the nodes of $\mathcal{G}_n$ a polynomial $\tilde{p}$ has to be selected which satisfies {eq}`orthcond`. Unless a suitable orthogonal basis of $\mathcal{P}_{n+1}$ is already known, this polynomial is obtained  with the Gram-Schmidt method: Start with the basis $\{1, x, x^2, \ldots, x^n, x^{n+1} \}$ of $\mathcal{P}_{n+1}$. The Gram-Schmidt method produces an orthonormal basis $\{ q_0, q_1, \ldots, q_n, q_{n+1} \}$ of $\mathcal{P}_{n+1}$. The function $q_{n+1}$ satisfies {eq}`orthcond`.
   
   Reason: $\{ q_0, q_1, \ldots, q_n\}$ is a basis of $\mathcal{P}_n$. Hence every $p \in \mathcal{P}_n$ can be written as $p = \sum_{i=0}^n \alpha_i q_i$ for some $\alpha_i \in \mathbb{R}$. Therefore
   
   $$
   \langle q_{n+1}, p \rangle = \langle q_{n+1}, \sum_{i=0}^n \alpha_i q_i \rangle = \sum_{i=0}^n \alpha_i \underbrace{\langle q_{n+1}, q_i \rangle}_{=0} = 0,
   $$
   
   proving {eq}`orthcond`.
2. Find the roots of $q_{n+1}$. They are the nodes of $\mathcal{G}_n$.
3. Calculate the weights of $\mathcal{G}_n$ with the formula
   
   $$
   \begin{eqnarray*}
   \sigma_{k} &=& \frac{1}{b-a} \int_a^b \prod_{m=0, \atop m \neq k}^n \frac{x-x_m}{x_k - x_m} \,
   \gamma(x) \, dx\\
   &=& \int_0^1 \prod_{m=0, \atop m \neq k}^n \frac{t-t_m}{t_k - t_m} \, \, \gamma((b-a)t+a) \, dt,
   \end{eqnarray*}
   $$
   
   where $t_m := (x_m - a)/(b-a)$.

Finding the roots of $q_{n+1}$ can be a difficult task if $n$ is large. There are special techniques to find roots of orthogonal polynomials, like recursion relationships. We will not cover them here, however most books on Gauss quadrature devote quite some time on this topic.

`````{dropdown} Theory of Gaussian quadrature

Gaussian quadrature is an advanced numerical method for evaluating integrals, particularly useful when dealing with functions that are difficult to integrate analytically. The general approach of Gaussian quadrature is based on the concept of weighted integrals, where the integration problem is defined as:

$$
\mathcal{I}(f) := \int_a^b f(x) \, \gamma(x) \, dx, \qquad f \in C[a,b],
$$

where $\gamma(x)$ is a weight function that simplifies the integration process for certain types of functions.

```{admonition} Understanding the role of the weight function $\gamma$
:class: note
1. **Practical Applications**: Often, one encounters integrals of functions $g(x)$ that can be decomposed into a product of a smoothly varying function $f$ and a weight function $\gamma$, such that $g = f \cdot \gamma$. In such cases, a common technique is to approximate $f$ by a polynomial $p$, and then integrate the product $p \cdot \gamma$.

2. **Extension to Unbounded Domains**: When extending this concept to unbounded integration domains, we consider $a = -\infty$ and $b = \infty$. Here, $\gamma$ is chosen to ensure that the product $p \cdot \gamma$ remains integrable for polynomials $p$ of degree $n$.

For initial understanding, consider the simpler case where $\gamma \equiv 1$ and $a, b \in \mathbb{R}$.
```

As before, we consider a quadrature formula of the form

$$
\mathcal{G}_n(f) = \sum_{k=0}^n \sigma_k f(x_k).
$$

In contrast to Newton-Cotes formulas, Gaussian quadrature benefits from the flexibility of choosing the nodes $x_k$. It is possible to significantly enhance the accuracy of the quadrature formula by optimising the node locations. We aim to achieve a quadrature method with a degree of exactness of $2n + 1$. This is facilitated by the concept of the scalar product:

$$
\langle f,g \rangle_{a,b,\gamma} = \int_a^b f(x) g(x) \, \gamma(x) \, dx,
$$

This function $\gamma$ is the same as the weight above and assumed to be piecewise continuous, integrable, and positive. Moreover, we assume that $\langle p, p \rangle_{a,b,\gamma}$ is finite for all $p \in \mathcal{P}_n$. Common examples include $\gamma = 1$ for $a,b \in \mathbb{R}$, $\gamma = 1/t$ for $a=0$, $b=\infty$, and $\gamma = 1/\sqrt{1-t^2}$ for $a=-1$, $b=1$.

Our focus is to demonstrate that the optimal quadrature nodes correspond to the roots of orthogonal polynomials. To understand this, we need to explore two key results about orthogonal polynomials.
 
```{prf:theorem} Roots of orthogonal polynomials
For a polynomial $\tilde{p} \in \mathcal{P}_{k+1}$, if

$$
\langle \tilde{p}, p \rangle = 0 \quad \forall \, p \in \mathcal{P}_k,
$$

then $\tilde{p}$ either equals zero or has exactly $k+1$ distinct roots in the open interval $(a,b)$.
```

```{prf:proof}
Suppose $\tilde{p} \neq 0$. If $\tilde{p}$ has roots $\xi_0, \xi_1, \ldots, \xi_\ell$ where its sign changes, then construct a polynomial $s(x) = \prod_{j=0}^\ell (x - \xi_j)$. If no such roots exist, let $s(x) = 1$. The product $\tilde{p} \cdot s$ does not change sign, implying $\langle \tilde{p}, s \rangle_{a,b,\gamma} \neq 0$. The degree of $s$ is greater than $k$, so $\tilde{p}$ has more than $k$ sign changes. However, as a polynomial in $\mathcal{P}_{k+1}$, $\tilde{p}$ can have at most $k+1$ sign changes.
```

This indicates that the roots of $\tilde{p}$ are ideal candidates for quadrature nodes. However, a question arises: is there a unique polynomial that satisfies this condition? If different polynomials meet the criterion, they could have varying roots, affecting the choice of nodes. The following theorem addresses this concern.

```{prf:theorem} Uniqueness of the orthogonal polynomial
For any two non-zero polynomials $\tilde{p}, \bar{p} \in \mathcal{P}_{k+1}$ satisfying the condition

$$
\langle \tilde{p}, p \rangle = 0 \quad \forall \, p \in \mathcal{P}_k,
$$

there exists a non-zero scalar $\alpha \in \mathbb{R}$ such that $\bar{p} = \alpha \, \tilde{p}$.
```

```{prf:proof}
Consider the polynomial representations of $\tilde{p}(x)$ and $\bar{p}(x)$. The coefficient $\tilde{a}_{k+1}$ must be non-zero; otherwise, $\tilde{p}$ would belong to $\mathcal{P}_k$. Setting $\alpha = \bar{a}_{k+1} / \tilde{a}_{k+1}$, we define $\Delta p = \alpha \tilde{p} - \bar{p}$. This difference falls within $\mathcal{P}_k$, and from our condition, it follows that 

$$
\| \Delta p \|^2 = \langle \Delta p, \Delta p \rangle = \langle \alpha \tilde{p} - \bar{p}, \Delta
p \rangle = \alpha \langle \tilde{p}, \Delta p \rangle - \langle \bar{p}, \Delta p \rangle = 0,
$$ 

implying $\bar{p} = \alpha \, \tilde{p}$.
```

```{prf:example} Orthonormal basis of $\mathcal{P}_2$
Using the Gram-Schmidt method, we construct an orthonormal basis for $\mathcal{P}_2$ with respect to the scalar product:

$$
\langle f,g \rangle_{-1,1,\gamma} = \int_{-1}^1 f(x) g(x) \, dx, \qquad \gamma = 1.
$$

The resulting basis functions are:

$$
q_0 = \frac{1}{\sqrt{2}},\quad q_1 = \sqrt{\frac{3}{2}} t,\quad q_2 = \sqrt{\frac{45}{8}} \Bigl(t^2 - \frac{1}{3}\Bigr).
$$

For $k=1$, $q_2$ meets the orthogonality condition, as any polynomial $p \in \mathcal{P}_1$ can be expressed as a linear combination of $q_0$ and $q_1$, leading to $\langle q_2, p \rangle = 0$. The roots of $q_2$ within $(-1,1)$ are $\pm \frac{1}{\sqrt{3}} \approx \pm 0.5774$. Other quadratic polynomials meeting this condition will be of the form $\delta (t^2 - 1/3)$ for some $\delta \in \mathbb{R}$.
```

We now arrive at the central result.
 
```{prf:theorem} Main theorem on Gaussian quadrature
For an integration weight $\gamma$, the interpolatory quadrature formula $\mathcal{G}_n$ achieves a degree of exactness $2n+1$ with respect to $\gamma$ if and only if its nodes $x_0, \ldots, x_n$ are such that the polynomial

$$
\tilde{p}(x) = \prod_{i=0}^n (x - x_i),
$$

meets the orthogonality condition $\langle \tilde{p}, p \rangle = 0 \quad \forall \, p \in \mathcal{P}_k$.
```

```{prf:proof}
a) If $\mathcal{G}_n$ has a degree of exactness $2n+1$ with respect to $\gamma$, then for all $f = \tilde{p} \cdot p \in \mathcal{P}_{2n+1}$ with $p \in \mathcal{P}_{n}$, it follows that

$$
\int_a^b \tilde{p}(x) \, p(x) \, \gamma(x) \, dx = \mathcal{G}_n (\tilde{p} \cdot p) = 0,
$$

satisfying the orthogonality condition.

b) Conversely, if the orthogonality condition holds, we show that $\mathcal{G}_n$ has a degree of exactness $2n +1$. For any $f \in \mathcal{P}_{2n+1}$, polynomial division yields $f = \tilde{p} \cdot p + r$ with $p, r \in \mathcal{P}_k$. Then

$$
\int_a^b f(x) \, \gamma(x) \, dx = \int_a^b \tilde{p}(x) \cdot p(x) \, \gamma(x) \, dx + \int_a^b r(x) \, \gamma(x) \, dx = 0 + \int_a^b r(x) \, \gamma(x) \, dx.
$$

Similarly, because $\tilde{p}(x_k) = 0$,

$$
\mathcal{G}_n(f) = \sum_{i=0}^n \sigma_k \bigl( \tilde{p}(x_k) \cdot p(x_k) + r(x_k) \bigr) = \sum_{i=0}^n \sigma_k r(x_k) = \mathcal{G}_n(r),
$$

leading to the conclusion that $\mathcal{G}_n$ integrates $f$ exactly, proving its degree of exactness is $2n+1$.
```


This concludes the theoretical foundation of Gaussian quadrature. Next, we explore its practical implementation.

To construct $\mathcal{G}_n$, follow these steps:

1. **Finding the Nodes**: Select a polynomial $\tilde{p}$ satisfying the orthogonality condition. If an orthogonal basis of $\mathcal{P}_{n+1}$ is not known, use the Gram-Schmidt method starting with the basis $\{1, x, x^2, \ldots, x^n, x^{n+1}\}$ of $\mathcal{P}_{n+1}$.
   
2. **Roots of $q_{n+1}$**: Determine the roots of $\tilde{p}$; these are the nodes of $\mathcal{G}_n$.

3. **Calculating Weights**: Compute the weights of $\mathcal{G}_n$ using the usual formula for interpolatory quadrature rules, involving the integration of a Lagrange basis polynomial.

```{note} 
Finding roots of $\tilde{p}$ for large $n$ can be challenging. Special techniques for finding roots of orthogonal polynomials, such as recursion relationships, are available but beyond the scope of this discussion. Most literature on Gaussian quadrature delves into these techniques in detail.
