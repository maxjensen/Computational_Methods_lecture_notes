# Principles of Backward Error Analysis

Modern error analysis traces its origins to the seminal work of [James Hardy Wilkinson](https://en.wikipedia.org/wiki/James_H._Wilkinson). It distinguishes between forward error, which is the error in the result of an algorithm, and backward error, which is a measure of perturbation in the input data.

In the following, we denote by $f: X \rightarrow Y$ a function that maps data from an input space $X$ to an output space $Y$. We assume that both $X$ and $Y$ are finite dimensional real vector spaces. Correspondingly, we define a function $\tilde{f}: X \to Y$, which represents an approximate evaluation of $f$. Consider, for example, the solution of the non-singular linear system of equations

$$
Ay = b
$$

with $A \in \mathbb{R}^{n \times n}$ and $b \in \mathbb{R}^n$. Then, the input space is the product space $X = \mathbb{R}^{n \times n} \times \mathbb{R}^n$ and our input data is the tuple $(A, b) \in X$. The output space is $Y = \mathbb{R}^n$. The function $f$ is given as

$$
f(A, b) = A^{-1}b
$$

and the approximate function $\tilde{f}$ is, for example, the Gaussian elimination algorithm in floating point arithmetic to compute the solution $y$.

## Forward Error

Let $y = f(x)$ and $\tilde{y} = \tilde{f}(x)$. The absolute forward error of $f$ at $x$ is defined as

$$
E_{abs} := \|y - \tilde{y}\|.
$$

The relative forward error $E_{rel}$ of $f$ at $x$ is defined as

$$
E_{rel} := \frac{\|y - \tilde{y}\|}{\|y\|}
$$

whenever $y \neq 0$.

## Backward Error

The backward error addresses the following question:

> Given an input $x$ and the corresponding output $\tilde{y} = \tilde{f}(x)$ from the *perturbed* function $\tilde{f}$, by how much would one need to perturb $x$ to an input $\tilde{x} = x + \Delta x$ so that the original function $f$ returns $\tilde{y}$?

In other words, we ask if the exact function $f$ could produce the same output as $\tilde{f}$ but with a slight perturbation in the input data. The idea is that our approximate function $\tilde{f}$ is useful if only a small perturbation is needed for the input data so that $f$ produces the same output as $\tilde{f}$. Note that the perturbation generally depends on the input $x$.

````{prf:example}
Let $f(x) = x^2 + x$ and $\tilde{f} = x$, its linearisation at $0$. Let $x = 1$ so that $\tilde{y} = 1$. The function $f$ attains $1$ at $\tilde{x}_{1,2} := - \frac{1}{2} \pm \frac{\sqrt{5}}{2}$.

Therefore, $\Delta x_1 := \tilde{x}_1 - x = (- \frac{1}{2} + \frac{\sqrt{5}}{2}) - 1$ and $\Delta x_2 := \tilde{x}_2 - x = (- \frac{1}{2} - \frac{\sqrt{5}}{2}) - 1$ are perturbations of $x$ to reproduce $\tilde{f}$'s output.
````

````{prf:example}
In the second example, we reverse the roles of $f$ and $\tilde{f}$. Let $f(x) = x$ and $\tilde{f} = x^2 + x$. Given $\tilde{y} = 1$, there are two possible inputs: $- \frac{1}{2} \pm \frac{\sqrt{5}}{2}$. Re-reading the question in the box above clarifies that we assume the choice of input is given: suppose it is the positive input $x = - \frac{1}{2} + \frac{\sqrt{5}}{2}$. Now, with $\tilde{x} = 1$, the perturbation is $\Delta x_1 := \tilde{x} - x = 1 - (- \frac{1}{2} + \frac{\sqrt{5}}{2})$.
````

The previous examples show how a potential lack of bijectivity of $f$ makes the backward error concept more complex than that of the forward error and that there may be many possible $\Delta x$ satisfying $\tilde{y} = f(x + \Delta x)$. We wish to find a smallest perturbation $\Delta x$, meaning it is obtained by solving a minimisation problem.

Formally, the absolute backward error of $f$ at $\tilde{y}$ with input $x \in \tilde{f}^{-1}(\tilde{y})$ is therefore defined as

$$
\eta_{abs}(\tilde{y}, x) := \min\{\epsilon \in [0, \infty): \, \exists \, \Delta x \in X \text{ such that } f(x+ \Delta x) = \tilde{y} \text{ and } \|\Delta x\|\leq \epsilon\}.
$$

Similarly, the relative backward error of $f$ at $\tilde{y}$ with input $x \in \tilde{f}^{-1}(\tilde{y})$ is defined as

$$
\eta_{rel}(\tilde{y}, x) := \min\{\epsilon \in [0, \infty): \, \exists \, \Delta x \in X \text{ such that } f(x+ \Delta x) = \tilde{y} \text{ and } \|\Delta x\|\leq \epsilon \|x\|\}.
$$

The definitions simplify if we assume that both $f$ and $\tilde{f}$ are invertible. Then there exists only one $x \in \tilde{f}^{-1}(\tilde{y})$ and we may drop the $x$ argument in $\eta_{abs}$ and $\eta_{rel}$. Furthermore, there is one $\tilde{x}$ such that $f(x+ \Delta x) = \tilde{y}$. The minimum of $\eta_{abs}$ is attained when $\|\Delta x\| = \epsilon$, that is

$$
\eta_{abs}(\tilde{y}) = \|\Delta x\| = \| x - \tilde{x} \|.
$$

Analogously, the minimum of $\eta_{rel}$ is attained when $\|\Delta x\| = \epsilon \|x\|$, that is

$$
\eta_{rel}(\tilde{y}) = \frac{\|\Delta x\|}{\|x\|} = \frac{\| x - \tilde{x} \|}{\|x\|}.
$$

Thus, for invertible $f$ and $\tilde{f}$, the absolute (resp. relative) backward error of $f$ at $\tilde{y}$ is equal to the absolute (resp. relative) forward error of $\tilde{f}^{-1}$ at $\tilde{y}$, now viewing $f$ as an approximation of $\tilde{f}$.

```{prf:example}
Let $f(A, b) = A^{-1} b$ as introduced in this section. Then, given a vector $\tilde{y}$, $f^{-1}(\tilde{y})$ is the set of matrix-vector pairs $(A,b)$ such that $A \tilde{y} = b$. Clearly, there are many linear systems that have the solution $\tilde{y}$ and therefore this $f$ is not invertible.
```

## Condition Number

The definition of the backward error may seem convoluted at first, but it is very useful, as the following will show. We assume momentarily that $f$ is differentiable. Owing to the mean-value theorem, we have for an $\alpha \in [0,1]$:

$$
\begin{aligned}
\|y - \tilde{y}\| &\stackrel{(1)}{=} \|f(x) - \tilde{f}(x)\|\\
&\stackrel{(2)}{=} \|f(x) - f(x + \Delta x)\|\\
&\stackrel{(3)}{\leq} \|f'(x + \alpha \Delta x)\|\cdot \| \Delta x\|
\end{aligned}
$$

Equality (2) is possible by choosing $\tilde{x} = x + \Delta x$ such that $\tilde{f}(x) = f(\tilde{x})$. Hence, $\Delta x$ is the backward perturbation associated with the output $\tilde{y}$. The right-hand side in (3) is the product of the derivative of $f$ and the size of the backward error. It demonstrates that the absolute forward error depends on two quantities: the sensitivity of $f$ to small perturbations (i.e. $\|f'(x)\|$) and the size of the absolute backward error $\|x-\tilde{x}\|$. This is of fundamental importance. We shall aim to devise an extremely good evaluation $\tilde{f}$ with a small backward error. The output forward error will still be large if the problem is highly sensitive to perturbations.

The corresponding statement for relative errors is based on the important definition of sensitivity given by the *condition number*. The relative condition number $\kappa_{rel}$ is defined as

$$
\kappa_{rel}(x) := \lim_{\delta\rightarrow 0} \sup_{\|\Delta x\|\leq \delta}\left(\frac{\|\Delta f\|}{\|f\|} \Bigl/ \frac{\|\Delta x\|}{\|x\|}\right)
$$

with $\Delta f = f(x) - f(\tilde{x})$ being the perturbation in the output data. Hence, it measures the largest possible ratio of relative output error with respect to relative input error under small perturbations. If $f$ is differentiable, you can show (self-check question)

$$
\kappa_{rel}(x) = \frac{\| x \|}{\|f(x) \|}\| f'(x) \|.
$$

From the definition of the condition number, you can show for differentiable $f$ that (self-check question)

$$
\frac{\|\Delta f\|}{\|f\|} \leq \kappa_{rel}(x)\cdot \frac{\|\Delta x\|}{\|x\|} + \text{higher-order terms}
$$

In other words, the relative forward error is bounded by the product of the condition number and the relative backward error.

What is a good condition number in practice? Ideally, we want the condition number to be as small as possible so that errors in the input data are not amplified in the output error. For each magnitude of the condition number, we lose one digit of accuracy in the output error. This can be seen as follows: Assume we work in double precision and that the algorithm has a backward error that is a small multiple of $\epsilon_{mach}$. Hence, if the condition number is $100$, we expect to lose two digits of accuracy in the forward error. If the condition number is in the order of $\epsilon_{mach}^{-1}$, then we will likely lose almost all digits.

If a condition number is small, we say that the problem is *well-conditioned*. If the condition number is large, we say that the problem is *ill-conditioned*. The precise meaning of these terms depends on the requirements of the application.

```{admonition} Worst-case Bound
The condition number is a worst-case bound. In practice, the actual forward error may be better than predicted by the condition number.
```

In the following, we will generally not use the subscript $rel$ for relative quantities. Unless otherwise stated, we always work with relative quantities.

````{admonition} **Question**
:class: tip
Let $x \in \mathbb{R}^2$ and $f(x) = x_1 - x_2$. Compute the $\infty$-norm condition number of $f$. For what inputs is the condition number large?

Hint: Use the expression for the condition number of a differentiable function.
````

````{dropdown} **Answer**
The Jacobian of $f$ is $J = \begin{bmatrix}1 & -1\end{bmatrix}$. Hence, $\|J\|_{\infty} = 2$. For the condition number, we obtain

$$
\begin{aligned}
\kappa &= \frac{\|x\|_\infty}{\|f(x)\|_\infty} \|J\|_\infty \\
    &= \frac{2 \max\left\{|x_1|, |x_2|\right\}}{|x_1 - x_2|}.
\end{aligned}
$$

The condition number is large if $x_1 \approx x_2$. This reflects the issue of cancellation errors. Consider two numbers $x_1$ and $x_2$ that agree to the first 5 digits and each of them is accurate to 7 digits. The difference between the two numbers will only be accurate to 2 digits since the first 5 correct digits cancel each other out.
````

````{admonition} **Question**
:class: tip
Let $f$ be differentiable. Show that

$$
\kappa_{rel}(x) = \frac{\| x \|}{\|f(x) \|}\| f'(x) \|.
$$
````

````{dropdown} **Answer**
The term on the right-hand side under the lim sup resembles a difference quotient (note the change from $\leq$ to $=$ in the second step):

$$
\kappa_{rel}(x) = 
\frac{\| x \|}{\| f(x) \|} \lim_{\delta\rightarrow 0} \sup_{\|\Delta x\|\leq \delta} \frac{\| f(x + \Delta x) - f(x) \|}{\| \Delta x\|} = \frac{\| x \|}{\| f(x) \|} \lim_{\delta\rightarrow 0} \sup_{\|\Delta x\| = \delta} \frac{\| f(x + \Delta x) - f(x) \|}{\| \Delta x\|}
$$

Because $X$ and $Y$ are finite-dimensional real vector spaces, we may identify them as $X = \mathbb{R}^n$ and $X = \mathbb{R}^m$. Because $f$ is differentiable at $x$ there is a function $h: \mathbb{R}^n \to \mathbb{R}^{m \times n}$ with $\lim_{\| \Delta x \| \to 0} h(x + \Delta x) = 0$ such that

$$
f(x + \Delta x) = f(x) + f'(x) \cdot \Delta x + h(x + \Delta x) \cdot \Delta x.
$$

Hence, using the triangle inequality,

$$
\frac{\| f(x + \Delta x) - f(x) \|}{\| \Delta x\|} = \frac{\| f'(x) \cdot \Delta x + h(x + \Delta x) \cdot \Delta x \|}{\| \Delta x\|} \leq \frac{\| f'(x) \cdot \Delta x \|}{\| \Delta x\|} + \| h(x + \Delta x) \|.
$$

Owing to the Cauchy-Scharz inequality, the supremum of $\| f'(x) \cdot \Delta x \|$ with maximum $\| f'(x) \|$ among all $\|\Delta x\| = \delta$ is attained for $\Delta x$ collinear to $f'(x)$. We deduce

$$
\lim_{\delta\rightarrow 0} \sup_{\|\Delta x\|\leq \delta} \frac{\| f(x + \Delta x) - f(x) \|}{\| \Delta x\|} = \| f'(x) \|.
$$
 
Now the result follows.
````

````{admonition} **Question**
:class: tip
Let $f$ be differentiable. Furthermore $\| x \| \neq 0$ and $\| f(x) \| \neq 0$. Show that

$$
\frac{\|\Delta f\|}{\|f\|} \leq \kappa_{rel}(x)\cdot \frac{\|\Delta x\|}{\|x\|} + \text{higher-order terms}
$$
````

````{dropdown} **Answer**
We multiple and divide by an auxiary term:

$$
\frac{\|\Delta f\|}{\|f\|} \leq \Bigl( \frac{\|\Delta f\|}{\|f\|} \Big/ \frac{\|\Delta x\|}{\|x\|} \Bigr) \frac{\|\Delta x\|}{\|x\|} \leq \frac{\|\Delta x\|}{\|x\|} \sup_{
    \| \Delta \tilde{x} \| \leq \| \Delta x \|} \Bigl( \frac{\| f(x + \Delta \tilde{x}) - f(x) \|}{\| f(x) \|} \Big/ \frac{\|\Delta \tilde{x}\|}{\|x\|} \Bigr) 
$$

Because $\| f(x) \| \neq 0$ and $f$ is differentiable, we may use the arguments of the previous self-check question to show that the function

$$
g(\Delta x) := \sup_{
    \| \Delta \tilde{x} \| \leq \| \Delta x \|} \Bigl( \frac{\| f(x + \Delta \tilde{x}) - f(x) \|}{\| f(x) \|} \Big/ \frac{\|\Delta \tilde{x}\|}{\|x\|} \Bigr)
$$

has the continuous extension 

$$
g(0) = \frac{\| x \|}{\|f(x) \|}\| f'(x) \| = \kappa_{rel}
$$

at $\Delta x = 0$, meaning that there is a function $\tilde{h}: \mathbb{R}^n \to \mathbb{R}$ with $\lim_{\| \Delta x \| \to 0} \tilde{h}(\Delta x) = 0$ such that

$$ 
g(\Delta x) = \kappa_{rel} + \tilde{h}(\Delta x).
$$

Now the result follows directly.
````
